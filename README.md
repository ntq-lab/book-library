# Package Manager
Use [YARN][yarn] to manage package

# Usage
Install [YARN][yarn]

Open command-line in project:
* Install package:
```bash
$ yarn install
```
* Run app:
```bash
$ npm start
```
Now browse to the app at [`localhost:9000`][local-app-url]
